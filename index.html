Instruction

  • 1. Search project address or click on the map
  • 2. Choose land use categories.
  • If the project has more than one categories, please sellect them all.
  • 3. Fill out Project Characteristics tab
  • Street Address of Project, Assessors Block/Lot and Transportation Analysis Zone Number will automatically update based on the location of your project.
  • 4. Select measures from each Category tab to achieve target associated with land use category.
  • Some measures, and options within measures, may be required by Planning Code. These measures are indicated.
  • Some measures have options. You must select "Yes" and the option presented to receive points.
  • Click on the name of measures for description of measure.
